# results/ — regenerated outputs

Everything here is regenerated by `python run.py`; nothing is hand-entered. Safe to
delete and rebuild.

| folder | produced by | contents |
|---|---|---|
| `bounds/` | `model/build_bounds.py` | per-gene and by-distance error-rate ceilings on the survival call; correlation-model stability |
| `strength/` | `model/build_strength.py` | the per-class OddsPath codes and their robustness |
| `evidence_codes/` | `model/build_evidence_codes.py` | **`evidence_codes.csv`** — one mechanism-labelled, transfer-adjusted code per held-out variant (the output of record); plus the mechanism-ablation table and a summary JSON |
| `reliability_map/` | `data_processing/reliability_map/` | where AlphaGenome is reliable enough to corroborate an RNA-route loss (needs the AG API; frozen copies shipped) |
| `validation/` | `validation/*.py` | the tables behind the four validation tests |
